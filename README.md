## Overview
This project demonstrates the use of various **transformer-based models** to perform key Natural Language Processing (NLP) tasks:
- Text Summarization using **BART (facebook/bart-large-cnn)**
- Response Generation using **OpenAI GPT**
- Interactive Chat with GPT
- Response Generation using **LLaMA (meta-llama/Llama-3.2-1B)**

Each task showcases practical implementations of prompt-based NLP, comparing different architectures and APIs for performance and fluency.
